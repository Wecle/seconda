# Agent 面试房间 UX 验证清单

1. 提交回答，确认蓝色候选人气泡先出现，随后才出现“面试官思考中”。
2. 确认新一轮思考自动展开，收到的是公开摘要而非模型原始推理，并在正式消息提交后自动折叠。
3. 手动展开或折叠已完成的思考面板；开始下一轮后，确认恢复自动展开模式。
4. 完成覆盖度持久化后确认显示“背景已保存”；刷新页面后卡片仍存在且不重复。
5. 强制触发 `blocking_limit`，确认临时问题消失、失败面板保持展开，SSE 不再自动重连。
6. 断开网络，确认最多自动重连五次，之后只显示手动重连入口。
7. 结束面试，确认依次显示评分进度、报告生成状态，只有 `completed` 后才能打开报告。
8. 在运行、评分、报告、失败各状态刷新页面，确认消息、卡片和进度均从持久数据恢复。
9. 检查 Network：初始详情 GET 无并发重复，终态后无持续 Run/SSE 请求。
